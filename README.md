
## 🖼️ Adding more photos
Drop new photos into `assets/img/` and reference them in `index.html`.

## 🚀 Deploy on Vercel
1. Push this folder to a GitHub repo
2. Go to vercel.com → New Project → Import repo
3. Leave all settings default → Deploy
